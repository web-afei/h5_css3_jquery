Metro Tooltip

Metro tooltip is a powerful Tool, to show information that you need for your users.

Tooltips can be awesome with cool animations like bounceIn, wobble, shake, tada, fadeInDown and much more! (Animations are not available on IE. It will display a soft fade in instead).

Also has positioning top, bottom, right, left. Don't worry to customize all the tooltips, you can set a default values for all of them or to specific ones.

What? do you want more? Ok, what if you can show a big map of your location and hold the tooltip until you close it?

Not enough?, Humm, looks like you know what you want. Let's show a youtube video.

Metro Tooltip is powerful, but also looks great on Macs with retina. And with an amazing icon set thanks to www.pixellove.com (free icon set), this works even better!, but you can use any kind of image.

Let's check some examples

  • A blogging tool... Wordpress
  • A voiceIP service... Skype
  • A video-sharing website... Youtube
